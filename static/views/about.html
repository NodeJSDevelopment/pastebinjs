About PasteBin.JS

“A pastebin is a type of web application where anyone can store plain text. They are most commonly used to share short source code snippets for code review via Internet Relay Chat.” - Wikipedia

The pastebin.js application was created by Petro Podrezo. It is open source under the MIT license which means you can download and host your own by going to GitHub and cloning yourself a copy. So why choose PasteBin.JS? PasteBin.JS has been designed with many significant improvements in mind over traditional pastebin websites.

Elegant, modern user interface

We're using the currently latest version of Bootstrap along with Angular.JS to bring you the most responsive and intuitive interface possible. We include a CodeMirror text editor for both viewing and editing code allowing you to have full color syntax highlighting even as you're updating a paste.

Fully documented REST API

Swing over to our API docs and you will see how easy it is to tie in your application with a pastebin backend. If you're a developer, you can use this to make posts to pastebin from your mobile app, the command line, or even your toaster.

Fully Free & Open Source

The application is made available with an MIT license meaning that you can basically do whatever you want* with it. Change it, mold it to be your own, and run it on whatever systems you like. Please note that certain components of the pastebin application may be under different licenses.

* Please don't use it for evil :)
